# C++ STL 教學

## STL基本介紹
- C++的STL(Standard Template Library) 是一套功能起i昂達的C++模版類和函數的結合，提供一系列通用的、可複用的常見資料結構與演算法
- STL的設計基於Template Programming，這意味著可以用於實作任何特定型別的程式
- STL包含多個組件，Container, Iterator, Algorithm, Function object, Adeptor, etc.
